Dustin Ingram

What's new in Python 3.7


Audience: All


Description:

# Description Python 3.7 is here! In this talk, we'll explore several new standard library modules, new syntax features and changes, and other performance improvements and implementation changes and what it means for us as Python developers. We'll also briefly chat about what exciting things didn't quite make it into 3.7, but we should expect to see in 3.8. # Outline * New things in Python 3.7 * New standard library modules * Dataclasses * History of dataclasses * The attrs project * Usefulness * Contextvars * Usefulness * Order-Preserving Dictonaries * History of Python Dictionary implementation * Recent changes * Usefulness * Modules supportings `__getattr__` and `__dir__` (PEP 562) * Usefulness (lazy-loading, deprecation) * The async and await keywords * PEP 563: postponed evaluation of type annotations * Upcoming in Python 3.8
